# 🐢 The Tortoise and the Hare

A small animated scene inspired by the classic story of *The Tortoise and the Hare*.

As the turtle slowly advances, the world gradually transforms from day into night. The sleeping hare remains unaware while the determined turtle continues its journey toward the finish line.

## ✨ Features

- Smooth day-to-night transition
- Animated sun, moon, clouds, and stars
- Interactive turtle movement
- Sleeping hare with animated eye effect
- Forest-themed environment
- Finish line and decorative elements

## 🎮 Controls

Hold the mouse button to help the turtle move forward.

## 🛠 Built With

- JavaScript
- p5.js
